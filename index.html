서버 연결 안내

Vercel 정적 사이트는 파일을 영구 저장하지 못합니다. 그래서 이 버전은 로컬 자동저장과 Supabase 서버 동기화를 같이 준비해뒀습니다.

  1. Supabase 프로젝트 생성
  2. supabase_setup.sql 실행
  3. config.js에 URL, anon key, 카카오 JavaScript 키 입력
  4. 재배포 후 서버 동기화 버튼 사용
